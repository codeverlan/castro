How to Find IntakeQ Selectors

  1. Open IntakeQ and navigate to a note form
  2. Right-click on the field you want to map
  3. Select "Inspect" to open Developer Tools
  4. Look for the field's id, name, or class attribute
  5. Use that as the selector:
    • ID: #field-name
    • Name: [name="field_name"]
    • Class: .field-class
